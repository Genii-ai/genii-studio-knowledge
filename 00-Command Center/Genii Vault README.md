# Genii Vault

The central knowledge base for Homegenii, FBX, and Genii Technologies.

## Structure

| Folder | Purpose |
|--------|---------|
| `00-Inbox` | Capture everything here first. Process daily. |
| `01-Projects` | Active projects — land deals, factory builds, platform dev |
| `02-Agents` | AI agent configs, SOUL files, run logs, capabilities |
| `03-ERPNext` | ERP configuration, customizations, SOPs |
| `04-Operations` | Factory ops, QC, labor coordination |
| `05-Finance` | Accounting, loans, investor relations, cap table |
| `06-Legal` | Contracts, compliance, HCD, entity docs |
| `07-Archive` | Completed projects, old notes, reference |
| `99-Daily-Notes` | Daily standups, meeting notes, thoughts |
