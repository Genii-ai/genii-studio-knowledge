# Genii Studio — AI Command Center

This vault is the canonical interface for Rufio (AI Lead Admin) and the Genii agent fleet.

## Structure

| Folder | Purpose |
|--------|---------|
| 00-Command Center | Dashboards, active sprints, current priorities |
| 01-Projects | Active project specs, plans, and decisions |
| 02-Agents | Agent configurations, skill mappings, runtimes |
| 03-Systems | Infrastructure docs, runbooks, architecture decisions |
| 04-Team-Memory | Team context, SOPs, recurring patterns |
| 05-Runbooks | Step-by-step procedures |
| 06-Scratch | Ephemeral notes, drafts |
| 99-Archive | Completed projects, old decisions |
| 99-Daily-Notes | Daily standups, journal |
| Templates | Reusable templates for plans, decisions, runbooks |

## Conventions
- All markdown output from Rufio lands here first.
- Use `[[WikiLinks]]` for cross-referencing.
- Tag decisions with `#decision`.
- Tag runbooks with `#runbook`.
