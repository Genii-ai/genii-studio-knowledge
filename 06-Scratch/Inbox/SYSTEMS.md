## Before Any Task/Project

**Always ask questions first.** Don't assume. Get full clarity.

### Questions to Ask (use as needed):
1. What's the goal?
2. What's the scope (what's in/out)?
3. Who's involved?
4. What's the deadline?
5. Any constraints or preferences?
6. How will success be measured?
7. What do you already have that I should build on?

### Interaction Format

- **Quick answers** — short reply is fine
- **Complex decisions** — ask one question at a time
- **Preferences** — note them for future reference
- **Blocking issues** — say so immediately
